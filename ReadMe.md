# A Random Mixed message.

This is a simple JavaScript program that generates a random javascript question and answer pair using a generateMessage function, and displays them on a webpage using DOM manipulation.

## Usage

To generate a question and answer pair, and display them on a webpage, open the index.html file in a web browser.

Click on the genearte message button, this will call the generateMessage function and use DOM manipulation to display a random question and answer on the webpage.

## Technologies Used

- JavaScript
- HTML
- CSS
